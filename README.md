## Overview

This repository contains **two complete music streaming applications** built with modern technologies:

### Mobile App (React Native + Expo)
- **Platforms**: Android, iOS (WIP)
- **Location**: `openspot-music-mobile/`
- **Features**: Native mobile experience with offline downloads, background playback, and persistent state

### Desktop App (Electron + React)
- **Platforms**: macOS, Windows (WIP), Linux (WIP)
- **Location**: `openspot-music-electron/`
- **Features**: Desktop-optimized interface with global audio controls and local storage

---

## Features

- **High-Quality Streaming**: Listen to your favorite tracks in the best possible quality
- **Offline Downloads**: Save music directly to your device for offline listening
- **Likes & Collections**: Build your personal music collection
- **Background Playback**: Continue listening while using other apps
- **No Login Required**: Jump right in without accounts or sign-ups
- **Persistent State**: Your liked songs and playback history saved across app restarts
- **Beautiful UI**: Clean, modern interface designed for each platform
- **Completely Free & Ad-Free**: Uninterrupted music without cost or advertisements

## Quick Start

### Mobile Development
```bash
git clone https://github.com/BlackHatDevX/openspot-music-app.git
cd openspot-music-app/openspot-music-mobile/
npm install
npx expo start
```

### Desktop Development
```bash
git clone https://github.com/BlackHatDevX/openspot-music-app.git
cd openspot-music-app/openspot-music-electron/
npm install
npm run electron-dev
```

### Build for Production

**Mobile:**
```bash
cd openspot-music-app/openspot-music-mobile/
eas build --platform android --profile production
```

**Desktop:**
```bash
cd openspot-music-app/openspot-music-electron/
npm run build
npm run electron-pack
```
